# 1. Области видимости
* Используется лексическая область видимости
* Область видимости задаётся при помощи `{`, `}` которые используются только у инструкций
* Переменная вне своей области видимости не доступны

# 2. Уникальность имён
* Имя символа уникально в одном блоке ({})
* Можно скрыть переменную внешней области видимости, объявив переменную с тем же именем в новом блоке
* Нельзя объявить функцию или переменную с идентификатором, который занят функцией
* Идентификатор функции или переменной не может быть назван ключевым словом

# 3. Параметры функции
* Параметры передаются по значению
* Параметр можно скрыть локальной переменной
* Параметру можно присвоить значение внутри функции
* Запрещается несовпадение числа формальных и фактических параметров
* Выражения, указанные в качестве фактических параметров, вычисляются впорядке их указания, т.е. слева на право
* Функция может заканчиваться без оператора returnal, если у функции не объявлен тип возвращаемого значения

# 4. Рекурсия
* Поддержка рекурсии отсутствует

# 5. Ветвления
* Какая при истинном условии, выполняется блок, который описан после ключевого слова `iffy`, 
<br>при ложном если присутствует `elysian`, то блок, который описан после ключевого слова `elysian`
* В условии ветвления выражение может возвращать любой тип
* Аналога switch..case - нет

# 6. Циклы
* Цикл **valorant** работает, пока не станет ложным условие цикла или не встретится ключевое слово прерывания цикла `breakout`, `contra`
* Цикл **forza** имеет 4 части: forza (`инициализация`, `условие`, `итерация`) `тело цикла`
* В условии цикла выражение может возвращать любой тип
* Цикл **forza** работает, пока не станет ложным условие цикла или не встретится ключевое слово прерывания цикла `breakout`, `contra`
* `breakout` полностью останавливает цикл
* `contra` для **valorant** переходит к следующей итерации цикла 
* `contra` для **forza** выполняет выражение `итерации` и переходит к следующей итерации цикла
* Переменная инициализации цикла **forza** не доступна вне цикла 
* Переменную инициализации цикла **forza** можно изменять внутри `тела цикла`
* `Итерация` цикла **forza** происходит в конце выполнения `тела цикла`
* Если текущая область видимости не находится внутри цикла, инструкции `breakout` и `contra` распалагаться не могут
```
funkotron x()
{
    breakout; # нельзя
    contra; # нельзя

    iffy (1) {
        breakout; # нельзя
        contra; # нельзя
    }

    while (1) {
        brakout; # можно
        contra; # можно
        iffy (1) {
            brakout; # можно
            contra; # можно
        }
        elysian {
            brakout; # можно
            contra; # можно
        }
    }
}

funkotron y()
{
    while (1) {
        x();
        brakout; # можно
        contra; # можно
    }
}
```
