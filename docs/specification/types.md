# 1. Краткое описание
## Типы
| Тип данных | Описание                 | Значение по умолчанию | Размер / Кодировка |
| ---------- | ------------------------ | --------------------- | ------------------ |
| dayzint    | Целочисленный тип данных | 0                     | 4 байта            |
| fallout    | Число с плавающей точкой | 0.0                   | 8 байт             |
| strike     | Строковый тип данных     | ""                    | UTF-8              |

## Ключевые особенности
1. При объявлении обязательно указывать тип переменной
2. Тип переменной после объявления изменить нельзя
3. Разрешены неявные преобразования для числовых типов данных
* Преобразование `dayzint` в `fallout` происходит с добавлением дробной части равной 0 (нулю)
* Преобразование `fallout` в `dayzint` происходит с откидыванием дробной части 
4. Разрешены неявные преобразования в логических выражениях
* В логических выражениях значение отличное от 0 (нуля) или "" (пустой строки) интерпретируется, как true, иначе false
* Логические выражения всегда возвращают `dayzint`: 0 или 1 
5. Явные преобразования в языке отсутствуют

## Функции работы со строками

| Сигнатура           | Описание                |
| --------------------| ----------------------- |
| length(str)         | Возвращает длину строки |
| char_at(str, index) | Извлекает отдельный символ из строки по указанной позиции <br> Возвращает строку из одного символа <br> При отсутствующем индексе - ошибка |

# 3. Операторы
* При вычислении выражения с разными числовыми типами `dayzint` преобразуется в `fallout`

## Арифметические операторы:
* Применять арифметические операции разрешено только для одинаковых типов, исключение - числовые типы данных

| Символы | dayzint                    | fallout                                     | strike       | 
| ------- | ---------------------------| ------------------------------------------- | ------------ |
| `+`     | Сложение                   | Сложение                                    | Конкатенация |
| `-`     | Вычитание или унарный "-"  | Вычитание или унарный "-"                   | Нет          |
| `*`     | умножение                  | Умножение                                   | Нет          |
| `/`     | Деление                    | Деление                                     | Нет          |
| `%`     | Остаток от деления         | Остаток от деления с дробной частью         | Нет          |
| `//`    | Целая часть от деления     | Целая часть от деления с дробной частью = 0 | Нет          |

## Логические операторы: 
* Все типы поддерживают все логические операторы описанные в [expression-grammar](./expression-grammar.md)
* Значение отличное от 0 (нуля) или "" (пустой строки) интерпретируется, как true
* Всегда возвращают dayzint: 0 или 1 

## Операторы сравнения:
* Все типы поддерживают все операторы сравнения описанные в [expression-grammar](./expression-grammar.md)
* Применять операции сравнения разрешено только для одинаковых типов, исключение - числовые типы данных
* Сравнение строк происходит по порядку кодов символов в кодировке
